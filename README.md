# astro-skelet

## Get Started

```
cd your-project

# get astro-skelet

git clone https://github.com/Selekkt/astro-skelet.git

# Install

npm i

# Run Site locally

npm start

# Build Site

npm run build
```

Navigate to `src/data/site.js` and edit to match your site's information.

Start writing new blog posts in Markdown at `src/pages/blog/`
